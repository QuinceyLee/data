某一个生产过程中的工艺描述 具体来讲 - 生产什么东西 - 需要什么原材料 原材料参数情况 - 需要哪些机器 机器监控哪些参数 - 机器组合方式即生产过程 - 初始状态下 机器控制设置参数 - 生产过程中需要几个大步骤 对应每个步骤过程中所获取到的传感器信息(包括生产正常和生产异常时，最好能够获取数据标签，例如在哪一个时间段生产出现了异常，异常种类，和相关传感器数据) 生产结果中 药品的质量评判标准，如何量化评价什么是黄金批次 另外针对某一生产过程中的常见几种异常情况的数据 包括 上述生产信息 以及那一个机器或者哪一个传感器发生了异常，出现了什么问题 导致了哪一结果变量的变化。 生产某种药品过程 1. 生产工艺介绍 - 即生产分为几个过程 - 需要哪些机器，机器怎么连接 机器详细信息 - 每个过程都发生了什么 - 中间产物信息 - 最终产物信息 - 介绍如何量化是否是异常情况 生产出现质量问题如何量化 是否能通过某一参数发现生产出现质量问题 2. 生产中的数据 - 每一批次对应的产品数据 包括产品编码、产品名称、产品类型、产品质量、产品质检信息 越多越好 - 生产所需要的工艺信息 包括工艺名称 工艺名称编号 工艺名称其它相关信息 - 生产中所需要的设备信息 设备名称 设备参数 设备控制参数 设备生产线号 越全越好 - 设备参数信息 包括 操作信息即如何控制机器运转的数据 工序-参数表，仪表实时得到的参数(例如：温度、ph等信息)，质检信息即每个步骤生产中间材料信息。 - 工艺设备参数对照信息 每个工艺需要哪些设备 每个设备在具体工艺中的设备参数信息 - 投料信息 每个工艺中所需要加入的原材料信息详情 包括 产品编码、批次、药材名称、投料量、投料时间等详细信息 3. 生产故障数据 - 这一生产中可能会出现哪些故障，故障原因有哪些，哪一步骤产生了故障，导致的结果数据(包含操作数据异常、观测到的设备数据异常、质量偏低)
